las instrucciones unarias son aquellas que tienen como **argumento una sola instrucci√≥n** por ejemplo tenemos `inc` que incrementa en 1 el valor del registro dado y `dec` que disminuye en 1 el valor del registro dado.

ejemplo:

supongamos que `rax = 2`

entonces:
```nasm
inc rax
```

ahora `rax` pasa a valer 3 (2 + 1).

y lo mismo ocurre con `dec`
supongamos que `rbx = 2`

```
dec rbx
```

en este caso `rbx` pasa a valer 1 (2 - 1)
