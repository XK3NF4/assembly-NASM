`mov` es una instruccion que seria como un copy es decir movemos el valor origen al destino sin modificar el origen.



ejemplo:



```nasm

mov rax, 0x1

mov rbx. rax

```



valor de `rax` es `0x1`

&nbsp;

valor de `rbx` es `0x1`
